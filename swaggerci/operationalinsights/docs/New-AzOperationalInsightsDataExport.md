# New-AzOperationalInsightsDataExport

## SYNOPSIS
Create or update a data export.

## SYNTAX

```
New-AzOperationalInsightsDataExport -Name <String> -ResourceGroupName <String> -WorkspaceName <String>
 [-SubscriptionId <String>] [-CreatedDate <String>] [-DataExportId <String>] [-DestinationResourceId <String>]
 [-Enable] [-LastModifiedDate <String>] [-MetaDataEventHubName <String>] [-TableName <String[]>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create or update a data export.
