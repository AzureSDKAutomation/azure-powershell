# New-AzOperationalInsightsLinkedService

## SYNOPSIS
Create or update a linked service.

## SYNTAX

```
New-AzOperationalInsightsLinkedService -Name <String> -ResourceGroupName <String> -WorkspaceName <String>
 [-SubscriptionId <String>] [-ProvisioningState <LinkedServiceEntityStatus>] [-ResourceId <String>]
 [-Tag <Hashtable>] [-WriteAccessResourceId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
